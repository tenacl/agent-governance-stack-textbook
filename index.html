The Agent Governance Stack — 개요

이 디렉토리는 Google Cloud Next 26 맥락의 Agent Governance Stack 글을 다산 공부법으로 재구성한 학습 패키지다.

파일 구성

  • textbook.html — 본 교재
  • references.md — 참고자료/학습 로드맵

읽는 순서

  1. 왜 에이전트 거버넌스가 필요한지 읽는다
  2. Identity → Tool Governance → Policy → Monitoring → Unified Posture 순으로 이해한다
  3. 마지막 챕터에서 내 시스템에 적용할 우선순위를 적는다

핵심 주장

에이전트 플릿은 SaaS 툴 모음처럼 관리하면 안 된다. 엔지니어링 조직처럼 정체성, 권한, 정책, 탐지, 감사 체계를 갖춘 운영 단위로 다뤄야 한다.
